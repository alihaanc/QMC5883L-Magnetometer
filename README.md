# QMC5883L-Magnetometer


> # Specification
 * Calibration and None calibration type
 
 * Filter support for smooth output
 
 * I2C Interrupt 
 
 * STM32 support
 
 * With python script
 
 * The moving average filter is a costly filter, don't forget to add a delay to your system.
